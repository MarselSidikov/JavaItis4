Git Bash - консоль (терминал). Интерфейс без интерфейса. 

cd - (Current Directory) - перейти в директорию
ls - (List) - список файлов и папок в текущей директории
ls -A - показать все файлы в текущей директории (включая скрытые)
git - программа, clone - команда, путь к репозиторию удаленному

TryGit - папка, внутри папки файл Text.txt

git status - команда проверки статуса репозитория локального
git add название_папки_или_файла - значит, подготовить файл для фиксации изменений

коммит - слепок изменений
git commit -m 'сообщение' - делает фиксацию изменений
